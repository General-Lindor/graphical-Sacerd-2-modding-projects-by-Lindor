Die so genannten "Cosmic Wings" sind ein Gegenstand aus Diablo 3.
Inspiriert von deren Design habe ich einen Vertex Shader programmiert,
welcher eine ingame-model-unabhängige 2-Dimensionale Oberfläche auf das Model projiziert.

Das Ergebnis sieht recht eindrucksvoll aus.
Es soll die die Texturen eines neuen Armorsets des Inquisitors verarbeiten.
